# AWS Landing Zone & Multi-Account Governance

## Overview
This project demonstrates a governance-focused AWS foundation using AWS Organizations, Service Control Policies (SCPs), IAM Identity Center, Control Tower, and CloudTrail.

## Current Scope
- Baseline SCP in CloudFormation YAML
- Multi-account governance starter structure
- Documentation for future expansion

## Repository Structure
- `cloudformation/` – CloudFormation YAML templates
- `docs/` – notes and architecture explanations
